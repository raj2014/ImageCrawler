
First create folders with names:

"output" and "photourls" in the C:\ drive


Execute the python code from command prompt:

python ImageCrawlerApp.py

App starts running with User Interface.

Follow the sequence of Operations:

CRAWL

ANALYZE METADATA

FETCH SIGHT LOCATIONS

LOAD IMAGES

Then we can see the photos getting collected in the output folder created in the C: drive
